プログラム復習コンテンツ

トゥイーニング（2点間の移動）

サンプルコードの結果

サンプルコード

　　 int startX=20;　　　//始点x座標

　　 int stopX=160;　　　//終点x座標

　　 int startY=30;　　　//始点y座標

　　 int stopY=80;　　　//終点y座標

　　 float x=startX;　　　//今のx座標

　　 float y=startY;　　　//今のx座標

　　 float step=0.005;　　　//ステップごとの移動量（0.0~1.0）

　　 float pct=0.0;　　　//移動量　百分率（0.0~1.0）

　　 　

　　 void setup(){

　　 　size(240, 120);

　　 }

　　

　　 void draw(){　　

　　 　background(0);

　　 　if(pct<1.0){

　　 　　x=startX+((stopX-startX)*pct);

　　 　　y=startY+((stopY-startY)*pct);

　　 　　pct+=step;

　　 　}

　　 　ellipse(x, y, 20, 20);

　　 }

説明文

　トゥイーニングとは画面上のある点（始点）から別の点（終点）へ図形が移動するアニメーションの処理のことを指している。

　始点と終点間の位置はフレームごとの計算で求める。

　初めに変数を設定しておくことで値の変更を容易にさせる。

　step変数の値を変更させると、移動スピードを変更させることができる。
